No: 01: What is ClassList.add() and ClassList.remove()?

According to my research jab hum ne Javascript k zariye koi new class add krwani ho to hum ClassList.add(' ') krty hen or parentheses me us class ka name likhty hen. or us class ki styling hum ne CSS me ki hoti hai.

Isi tran jab class remove krwani ho to ClassList.remove(' ') likhty hen or parentheses k ander usi class ka name likhty hen.

For example: signInBtn.classList.add('disable');

Note: ab yahan disable class ka name hai.

No: 03: How to Open a New Window for a Web Page in JavaScript?

Hum ne jis page ko new window me open krwana ho us ko alg se HTML me design kr k then usy JavaScript me window.open('jo page open krwana uska link yahan den gen.');

or hum new window ko apni mrxi se width and height b define kr skty hen.

agr hum ne current window me rehty hvy new page ko open krna ho to us ka b same hi treqa hai bus ,'_self' likhen gen.

For example: window.open(' insert page link here', '_self ');
